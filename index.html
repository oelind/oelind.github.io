This was a mokuhanga print project which is a Japanese style of print making. I also used wood blocks for This project (not certain on the type of wood). This print has four layers and is of a leaf I took a photo of this past fall.




A reduction print with the them being earth as an element I used my own drawings with some inspiration from actual fossils and rock layers. For a reduction print the same block is used for all of the layers which means the other layers (not including the final one) cannot be re-done or printed again as more of the material of the block has been carved away. In other words the editions of a reduction print are rather exclusive. I did three layers and used pine plywood.




This project the theme was to do a nature themed/ inspired pattern and based off of some leaves I used my sketch to design the print. This was a single layer print and I used a linolium block.
